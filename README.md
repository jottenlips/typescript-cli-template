# Typescript Command Line Tool Template

### Run Locally

```console
$ yarn
$ yarn compile
$ npm link
```


### Install 

```console
$ yarn global add your-package
```

### Testing & Linting
```console
$ yarn run test
```

```console
$ yarn run fix
```

### Use 

```console
$ your-command help                                                                                     
Options:
  --help     Show help                                                 [boolean]
  --version  Show version number                                       [boolean]
  --hello                           [string] [default: "Hello, no commands yet"]
```

```console
$ your-command --hello Hi!                                                                                    
Hi!
```
